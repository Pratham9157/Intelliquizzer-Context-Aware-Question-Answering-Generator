# 🧠 PART 1 — DISTRACTOR GENERATION

---

## 🔹 Function 1: WordNet-Based Distractors

```python
get_wordnet_distractors(answer: str, answer_type: str, n=3) -> List[str]
```

---

### Logic:

* Use `nltk.corpus.wordnet`
* Get:

  * synonyms
  * hypernyms → hyponyms
* Extract candidates

---

### Rules:

* Exclude:

  * exact answer
  * duplicates
* Clean:

  * replace "_" with space
* Return top `n`

---

---

## 🔹 Function 2: Context-Based Distractors (IMPORTANT)

```python
get_context_distractors(answer: str, context: str, answer_type: str) -> List[str]
```

---

### Logic:

* Use spaCy:

  * extract entities from context
* Select entities:

  * SAME TYPE as answer
* Remove:

  * correct answer

---

### Example:

Context:
"Einstein and Newton contributed to physics."

Answer: Einstein

Distractors:
Newton, Tesla, Galileo

---

---

## 🔹 Function 3: Hybrid Distractor Generator (MAIN FUNCTION)

```python
generate_distractors(answer: str, context: str, answer_type: str) -> List[str]
```

---

### Strategy:

1. Get:

   * context distractors
   * wordnet distractors

2. Merge them

3. Remove duplicates

4. Ensure:

   * exactly 3 distractors
   * if not enough → add fallback:

     ```python
     ["Option A", "Option B", "Option C"]
     ```
