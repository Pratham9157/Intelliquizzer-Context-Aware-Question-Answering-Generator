### 1. TF-IDF Vectorization

### 2. Document Centroid Calculation

Implement:

```python
compute_centroid(tfidf_matrix) -> np.ndarray
```

Logic:

* Mean of all sentence vectors

---

### 3. Sentence Scoring

Implement:

```python

```

Use:

* Cosine similarity between each sentence vector and centroid

---

### 4. Named Entity Boost (IMPORTANT FEATURE)

Use spaCy (`en_core_web_sm`)

Implement:

```python
ner_boost(sentence: str, base_score: float) -> float
```

Logic:

* Count number of named entities
* Add small boost:

```python
boost = 0.05 * num_entities
```

---

### 5. Final Ranking Function

Implement:

```python
rank_sentences(sentences: List[str], top_k: int = 10) -> List[Tuple[str, float]]
```

Pipeline:

1. TF-IDF transform
2. Compute centroid
3. Compute cosine similarity scores
4. Apply NER boost
5. Sort by score (descending)
6. Return top_k sentences (preserve original order optionally)
